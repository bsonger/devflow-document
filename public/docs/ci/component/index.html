一、CI 核心编排与触发#

组件 云原生领域 解决的问题 架构定位 是否必选 可替代方案

组件 云原生领域 解决的问题 架构定位 是否必选 可替代方案

代码规范 / Lint Quality 代码风格、规范检查 Quality Gate ❌ golangci-lint、eslint

组件 云原生领域 解决的问题 架构定位 是否必选 可替代方案
